# CV en ligne – Oumaima Zayane

## Description du projet
Ce projet consiste à créer un CV en ligne responsive avec HTML, CSS et Bootstrap, puis à le publier gratuitement avec GitHub Pages.

## Technologies utilisées
- HTML5
- CSS3
- Bootstrap 5
- GitHub Pages

## Structure du projet
- index.html
- css/style.css
- img/photo.jpg
- assets/cv.pdf

## Sections du site
- Accueil / Profil
- À propos
- Compétences
- Projets
- Formation & Expériences
- Contact
